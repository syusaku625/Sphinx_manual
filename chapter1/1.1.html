Sphinxのインストール～プロジェクト作成¶

実行環境¶

  • WSL(Windows Sysytem for Linux)

Sphinxのインストール¶

$ pip install sphinx

プロジェクトの作成¶

$ mkdir sample_project
$ cd sample_project
$ sphinx-quickstart
実行すると以下のような画面になる．
project nameとauthor name と project releaseとlanguageを入力する必要がある．
ここでは以下のように入力した
